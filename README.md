## 代码环境

## 代码运行(理论流程，未经过测验)

- **Step1**: 修改`run_trainer.py`中`Config`参数为`./config/acl.yaml`
- **Step2**：配置`./config/acl.yaml`文件中的参数
    - `data_root`: 数据集路径
    - `save_path`: 日志保存路径
- **Step3**: 运行代码`python run_trainer.py`
- **Step4**：日志保存在配置文件中`save_path`路径下
